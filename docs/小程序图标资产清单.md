# 小程序图标资产清单

## 本轮结论

小程序原先主要依赖文字、汉字缩写和纯文本按钮表达功能，缺少稳定的视觉识别。已补齐一套适合适老化健康记录工具的本地图标，统一放在 `assets/icons/`，并接入关键页面。

本轮没有直接下载第三方 iconfont 文件。原因是工具侧访问 iconfont 页面不稳定，且 iconfont 下载通常涉及页面交互、登录和授权确认。为避免引入授权不清晰的第三方 SVG，本轮改为按 iconfont 常见“线性矢量、圆角、简洁识别”的方向生成本地原创 PNG 资产。后续如要替换为 iconfont 官方图标，可按本文的关键词和风格约束重新挑选。

## 风格约束

- 视觉方向：圆角线性矢量图标，透明背景，少细节，高识别度。
- 主色：选中和主操作使用 `#168957`，普通灰使用 `#65756D`，危险/失败使用 `#C8463A`。
- 尺寸：tabBar 使用 81px PNG；页面内功能图标保持 48rpx 到 96rpx 的可视尺寸。
- 适老化：避免过细线条、复杂透视、低对比渐变和仅靠颜色表达状态。
- 合规边界：避免使用医院十字、医生、处方、诊断、急救等容易被误判为医疗诊断/问诊能力的图标。

## 页面接入范围

- tabBar：首页、趋势、家庭、我的已接入未选中/选中图标。
- 首页：记血压、记血糖、用药快捷入口已接入业务图标。
- 家属首页：趋势、用药、周报快捷入口已接入业务图标，空用药状态已接入空状态图标。
- 隐私确认页：基础资料、健康记录、提醒订阅、家庭授权已接入图标。
- 角色选择页：本人使用、帮家人管理已接入角色图标。
- 提醒设置页：提醒项目、提醒时间、夜间免打扰已接入图标。
- 隐私与授权页：授权项目和管理入口已接入图标。
- 数据管理页：数据范围、导出备份、删除记录、注销账号已接入图标。
- 我的页：设置入口、支持入口、账号入口、重建统计已接入图标。
- 帮助中心：隐私政策、用户协议、提醒设置、数据管理快捷入口已接入图标。
- 空状态：家庭成员、家属用药、用药提醒、用药计划、历史记录已接入统一空状态图标。
